# 自动抢红包说明
- **若有疑问请联系我** **联系方式TG:@ljh8686**

## 🚀 开启步骤
1. 任意聊天框发送“**红包设置**”进入配置页面
2. 找到并开启“**开启自动抢红包**”开关
3. 根据您的需求，完成各项细化设置并点击“**保存配置**”

## ⚙️ 功能详解
| 功能模块 | 说明 |
|:----------|:------|
| **自动抢红包** | 开启后自动识别红包消息并执行抢红包流程 |
| **自己红包控制** | 可设置“**不抢自己发的红包**”，按需启用 |
| **自动关闭页面** | 抢完后自动关闭领取页/详情页，减少停留 |
| **名单策略** | 支持三种模式：抢所有、仅白名单、拒抢黑名单（支持好友/群聊/手动ID） |
| **关键词过滤** | 支持“不启用 / 必须包含 / 包含则不抢”三种关键词策略 |
| **抢包延迟** | 支持自定义延迟数值与单位（毫秒/秒） |
| **自动回复** | 抢到红包后可自动回复，支持模板随机发送（用 `|` 分隔） |
| **回复延迟** | 支持随机延迟（0-2秒）或自定义延迟，降低行为一致性 |
| **日志开关** | 可开启/关闭日志输出，方便排查问题 |
| **统计信息** | 显示成功次数、累计金额、失败次数、今日次数，并支持一键重置 |

## 🧩 名单管理使用方法
1. 进入“**名单策略**”模块
2. 选择模式：
   - **抢所有红包（默认）**
   - **仅抢白名单**
   - **拒抢黑名单**
3. 点击“管理白名单/管理黑名单”后可选择：
   - 从好友列表选择
   - 从群聊列表选择
   - 手动输入微信ID（多个用逗号分隔）
   - 查看当前名单并删除/清空

## 🤖 自动回复说明
- 开启“抢到红包后自动回复”后生效
- 回复内容支持多模板，例如：`谢谢老板|感谢老板|老板大气`
- 系统会随机选择一条发送
- **自己发的红包不会触发自动回复**

## 📌 注意事项
1. 请确保模块已正确加载且微信进程已生效
2. 若提示“请关闭LSPosed调用保护”，请先按提示处理
3. 不同微信版本按钮字段可能变化，建议开启日志观察运行状态
4. 如出现红包页面启动失败，可稍后重试并检查版本兼容性
5. 建议先用小号或测试群验证配置，再正式使用
