# 微信AI回复插件

## 简介

为您的微信接入AI，实现智能私聊回复功能。
本插件由AI辅助制作，通过调用硅基流动的API实现AI回复功能
若需使用其他API服务商，请自行替换API地址和密钥。

## 功能

· AI私聊回复：自动响应私聊文本消息，调用AI生成回复。
· 上下文记忆：为每个用户独立保留多轮对话历史。
· 角色扮演：可自定义角色设定，让AI按设定风格回复。
· 对话重置：发送指定指令（默认指令为 /新对话）清空历史，开启新话题。
· 引用回复：可选择是否引用用户消息进行回复。
· 灵活配置：支持修改API密钥、模型、温度、最大token等参数。
· 多用户隔离：不同用户的对话历史和上下文互不干扰。
· 后台回复：微信在后台不影响AI回复。

支持WAuxiliary 1.2.7版本

## 用法

1. 将zip文件解压至/Android/media/com.tencent.mm/WAuxiliary/Plugin/
2. 在main.java文件代码开头的 API_KEY 处填入您在硅基流动申请的API密钥。
3. （可选）在main.java文件中自定义配置
4. 在WAuxiliary中打开插件开关
