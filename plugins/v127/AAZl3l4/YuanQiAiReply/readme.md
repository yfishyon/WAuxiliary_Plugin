## 使用方法

1. 登录元启AI平台，按指示配置角色人设
2. 登录元启AI平台，按指示获取ApiKey
3. 聊天框输入`/元启` 打开脚本配置页面 点击设置图标 配置API Key和默认回复词
4. 添加触发规则（关键词/正则/艾特）
5. 在需要开启的群/私聊中开启开关
6. 不懂进群看群公告
